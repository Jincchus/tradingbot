# 매매 전략 정의

각 전략은 독립된 Alpaca 페이퍼 계정에서 실행되며, 동일한 `BaseStrategy` 인터페이스(`select_symbols`, `on_bar`)를 구현한다. 모든 전략은 `_bar_buffer`(종목별 종가 롤링 버퍼)만으로 지표를 계산하므로 실시간 추가 API 호출이 없다.

공통 감시 종목: `AAPL, MSFT, NVDA, TSLA, GOOGL`
공통 포지션 크기: 예산의 20% (`POSITION_SIZE = 0.2`)

---

## v1 — MA 크로스오버 (Moving Average Crossover)

**유형:** 추세추종 (Trend Following)
**파일:** `strategies/ma_crossover.py` / `strategy_type = "ma_crossover"`

가장 기본적이고 널리 쓰이는 추세추종 전략. 단기 이동평균이 장기 이동평균을 돌파하는 시점을 추세 전환 신호로 본다.

**지표**
- 단기 이동평균 SMA(10)
- 장기 이동평균 SMA(30)

**매매 규칙**
- **매수(골든크로스):** SMA(10)가 SMA(30)를 상향 돌파 + 미보유 시
- **매도(데드크로스):** SMA(10)가 SMA(30)를 하향 돌파 + 보유 시 전량 청산

**특징**
- 강한 추세장에서 유리, 횡보장에서는 잦은 손실(휩쏘) 발생
- 신호가 늦게 나오는 후행 지표

## v2 — RSI 평균회귀 (RSI Mean Reversion)

**유형:** 평균회귀 (Mean Reversion)
**파일:** `strategies/rsi_reversion.py` / `strategy_type = "rsi_reversion"`

과매도 구간에서 매수, 과매수 구간에서 매도하는 역추세 전략. "오른 건 내리고 내린 건 오른다"는 평균회귀 가정에 기반한다.

**지표**
- RSI(14) — Relative Strength Index
- 계산: `RSI = 100 - (100 / (1 + RS))`, `RS = 평균상승폭 / 평균하락폭` (14봉 기준)

**매매 규칙**
- **매수:** RSI < 30 (과매도) + 미보유 시
- **매도:** RSI > 70 (과매수) + 보유 시 전량 청산

**특징**
- 횡보장·박스권에서 유리
- 강한 하락 추세에서는 "떨어지는 칼날 잡기" 위험 (과매도가 더 과매도로)
- 필요 버퍼: 최소 15봉

## v3 — MACD 시그널 크로스 (MACD Signal Crossover)

**유형:** 모멘텀 (Momentum)
**파일:** `strategies/macd.py` / `strategy_type = "macd"`

추세의 강도와 방향을 동시에 보는 모멘텀 전략. MA 크로스오버보다 반응이 빠르고 노이즈에 덜 민감해 가장 인기 있는 지표 중 하나.

**지표**
- MACD Line = EMA(12) − EMA(26)
- Signal Line = EMA(9) of MACD Line
- (Histogram = MACD − Signal, 참고용)

**매매 규칙**
- **매수:** MACD Line이 Signal Line을 상향 돌파 + 미보유 시
- **매도:** MACD Line이 Signal Line을 하향 돌파 + 보유 시 전량 청산

**특징**
- 지수이동평균(EMA) 사용으로 단순 SMA보다 최근 가격에 민감
- 추세 초입을 비교적 빠르게 포착
- 필요 버퍼: 최소 35봉 (EMA26 + Signal9 안정화)

## v4 — 볼린저 밴드 (Bollinger Bands Mean Reversion)

**유형:** 평균회귀 (Mean Reversion) / 변동성
**파일:** `strategies/bollinger.py` / `strategy_type = "bollinger"`

가격이 통계적 정상 범위를 벗어나면 다시 평균으로 회귀한다는 전략. 변동성(표준편차)을 밴드 폭으로 활용한다.

**지표**
- 중심선 = SMA(20)
- 상단 밴드 = SMA(20) + 2 × 표준편차(20)
- 하단 밴드 = SMA(20) − 2 × 표준편차(20)

**매매 규칙**
- **매수:** 종가가 하단 밴드 이하로 터치/이탈 + 미보유 시
- **매도:** 종가가 중심선(SMA20) 이상으로 회귀 + 보유 시 전량 청산

**특징**
- 박스권·변동성 장세에서 유리
- 밴드 돌파(추세 시작) 시에는 역방향 진입으로 손실 가능
- 필요 버퍼: 최소 20봉

## 비교 요약

| | v1 MA크로스 | v2 RSI | v3 MACD | v4 볼린저 |
|---|---|---|---|---|
| 유형 | 추세추종 | 평균회귀 | 모멘텀 | 평균회귀 |
| 유리한 장세 | 강한 추세 | 횡보 | 추세 초입 | 박스권 |
| 약점 | 횡보장 휩쏘 | 추세장 역행 | 횡보장 거짓신호 | 돌파장 역행 |
| 지표 후행성 | 높음 | 낮음 | 중간 | 중간 |
| 필요 버퍼(봉) | 31 | 15 | 35 | 20 |

서로 다른 시장 가정(추세 vs 평균회귀)을 가진 전략들을 동시에 돌려, 어떤 전략이 실제 시장에서 가장 좋은 성과를 내는지 페이퍼 트레이딩으로 비교한다.
